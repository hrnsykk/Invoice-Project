## About Invoice Project with Laravel and Inertia 

PDF Invoices creation application built using Laravel and Inertia.Js.


## Features
- Customer management
- Product management
-  PDF output


## Install

`
$ git clone https://github.com/hrnsykk/Laravel-Invoice-Project-with-Inertia.JS.git
`

`$ composer install`

`$ npm install `

`$ npm run dev`

`$ php artisan serve`
